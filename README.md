# LLM IA Soberana & Vibe Coding

## Objetivo
Comprender y aplicar los principios de la soberanía tecnológica desplegando un Modelo de Lenguaje Grande (LLM) de forma local.  
Utilizar el paradigma de **Vibe Coding** (programación asistida por IA) para crear un script funcional interactuando exclusivamente con herramientas de código abierto.

---

## Organización del Equipo
Este proyecto se desarrolla en modalidad grupal (3 integrantes), simulando el flujo de trabajo de un proyecto de Software Libre mediante ramas y Pull Requests.

- **Integrante 1 – Maintainer (Mantenedor):**  Gisella Henriquez
  Responsable de crear el repositorio, inicializar este README.md y gestionar las fusiones (merges) de los PR.  

- **Integrante 2 – SysAdmin (Operador de Infraestructura):**  Agustin Diumacan
  Encargado de levantar el entorno contenedorizado con Podman y documentar los comandos utilizados. Trabaja en la rama `feature/infra`.  

- **Integrante 3 – Vibe Coder:**  Brenda Uichaques
  Responsable de interactuar con la IA para generar el código automatizado, sin escribirlo a mano. Trabaja en la rama `feature/scripting`.  

---

## Flujo de Trabajo con Git
- Cada integrante trabaja en su rama (`feature/infra`, `feature/scripting`).  
- Los cambios se integran a `main` mediante Pull Requests revisados por el Maintainer.  
- Se verificará el historial de commits para comprobar el uso correcto de ramas y PR.
